<h1>This was made for a hackclub event</h1>
<p>This was made for horizons</p>
<p>This was made with no AI</p>
<h3>Movement</h3>
<p>Use either space or left click for movement, hold them down</p>
<p>The purpose of this game is to fly your paper aeroplane through the tunnel and achieve a grade</p>
<p>The grading system is based on NCEA standard</p>
<p>This game is called swivle game</p>
<p>A small issue you may encounter is that your speed carrys into when you die</p>
<p>You can run this game on my itch.io</p>
